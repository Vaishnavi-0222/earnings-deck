Code Sample: Inline P&L Transformation

import pandas as pd

# Load quarterly P&L (CSV export from EDW)
df = pd.read_csv("pl_q.csv")

# Normalize column names, compute YoY and margin
cols = {"revenue":"Revenue", "opex":"OpEx", "net_income":"NetIncome"}
df = df.rename(columns=cols)
df["OpMargin"] = (1 - df.OpEx / df.Revenue) * 100

df["YoY_Rev_%"] = df.Revenue.pct_change(periods=4).mul(100)
print(df.tail(4))

Discounted Cash Flow (DCF):

\[ PV = \sum_{t=1}^{T} \frac{CF_t}{(1+r)^t} \]

Portfolio Volatility:

\[ \sigma_p = \sqrt{\mathbf{w}^\top \Sigma \, \mathbf{w}} \]
